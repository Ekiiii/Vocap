# VOCAP

Logiciel de Bande Rythmo open-source pour tous les fandubers.

VOCAP est une application moderne conçue pour faciliter la création et l'édition de bandes rythmo.

## Caractéristiques

- **Bande Rythmo Interactive** : Visualisation en temps réel avec défilement fluide.
- **Éditeur de Dialogue** : Interface intuitive pour saisir et synchroniser vos textes.
- **Support Multi-Pistes** : Importez plusieurs fichiers audio et gérez les volumes.
- **Export Haute Qualité** : Exportez vos sessions avec incrustation de la bande rythmo.
- **Multi-Fenêtre** : Sortie vidéo secondaire pour un setup double écran.

## Installation

```bash
# Clonez le dépôt
git clone https://github.com/votre-utilisateur/vocap.git

# Allez dans le dossier app
cd vocap/app

# Installez les dépendances
npm install
```

## Développement

```bash
# Lancez l'application en mode développement
npm run dev
```

## Build

```bash
# Créez l'installateur Windows
npm run dist
```

## Technologies

- **Frontend** : React, TypeScript, Vite, TailwindCSS, Framer Motion.
- **Backend** : Electron, Node.js.
- **Audio/Vidéo** : FFmpeg.
